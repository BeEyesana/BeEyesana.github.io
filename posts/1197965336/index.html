输入

使用 Scanner 类来获取输入

创建 Scanner 对象要传入 InputStream 对象 System.in

Scanner 类默认的分隔符就是空格

使用

通常使用 while(sc.hasNextLine()) 或者 while(sc.hasNext()) 来读取多行输入数据

  • hasNext()：判断扫描器中当前扫描位置后是否还存在下一段。
  • hasNextLine()：如果在此扫描器的输入中存在另一行，则返回 true

获取数据使用 sc.next() 和 sc.nextLine() 方法，它们的返回值都是 String

next() 与 nextLine() 区别：

  1. next()
  • 一定要读取到有效字符后才可以结束输入
  • 对输入有效字符之前遇到的空格，next() 方法会自动将其去掉
  • 只有输入有效字符后才将其后面输入的空格作为分隔符或者结束符
  • next() 不能得到带有空格的字符串。
  1. nextLine()
  • 以 Enter 为结束符,也就是说 nextLine() 方法返回的是输入回车之前的所有字符。
  • 可以获得空格

总结：next() 碰到空格、换行都结束输入；而 nextLine() 只以换行（回车）结束输入。

输出

输出使用 System.out.print() 等方法

print()、println() 和 printf() 之间的区别

  • print() ：打印数据。
  • println()：打印数据，并且使光标移动到下一行的开头。
  • printf()：打印格式化的数据（类似于C / C ++编程中的 printf ）。

格式化输出

通过使用占位符 %?，printf() 可以把后面的参数格式化成指定格式

占位符 说明
%d 格式化输出整数
%x 格式化输出十六进制整数
%f 格式化输出浮点数
%e 格式化输出科学计数法表示的浮点数
%s 格式化字符串
